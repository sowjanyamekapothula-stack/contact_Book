# 📒 Contact Book Management System

A simple Contact Book application built using Python and Streamlit. This project helps users manage contact details efficiently through a user-friendly web interface.

---

## 🚀 Features
- Add new contacts
- View saved contacts
- Search contacts
- Update contact details
- Delete contacts
- Attractive user interface

---

## 🛠️ Technologies Used
- Python
- Streamlit
